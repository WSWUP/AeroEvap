
AeroEvap- Aerodynamic Mass-Transfer Evaporation
===============================================

.. image:: https://readthedocs.org/projects/aeroevap/badge/?version=latest
   :target: https://aeroevap.readthedocs.io/en/latest/?badge=latest
   :alt: Documentation Status

Tools for applying the aerodynamic mass-transfer approach to estimate open-water evaporation. ``AeroEvap`` has methods for parallelizing the calculations on a time series of meterological input data that is loaded as a :obj:`pandas.DataFrame`.

`View on GitHub <https://github.com/WSWUP/AeroEvap>`_.
